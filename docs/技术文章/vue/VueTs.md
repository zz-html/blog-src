demo源码地址 https://github.com/zz-html/vue3-ts-project/tree/main/vue-ts-demo
## 创建工程
```
pnpm create vite
```

输入vue项目名称，vue-ts-demo  
选择Vue  
选择TypeScript  
cd 进入到 vue-ts-demo 项目路径下
```
pnpm install
pnpm run dev
```

浏览器打开 http://127.0.0.1:5173/
